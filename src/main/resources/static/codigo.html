Variables en Java

Una variable es una ubicación en la memoria de la computadora, en donde se puede guardar un valor para utilizarlo posteriormente en un programa. Todas las variables deben declararse con un nombre y un tipo antes de poder usarse; este nombre permite al programa acceder al valor de la variable en memoria; y puede ser cualquier identificador válido. El tipo de una variable especifica el tipo de información que se guarda en esa ubicación de memoria. Al igual que las demás instrucciones, las instrucciones de declaración terminan con punto y coma (;). 


int numero1;
int numero2;

Las instrucciones de declaración de variables declaran que las variables numero1 y numero2 contienen datos de tipo int; estas variables pueden contener valores enteros (números completos, como 7, –11, 0 y 31,914). Estas variables no se han inicializado todavía. El rango de valores para un int es de –2,147,483,648 a +2,147,483,647: 


int numero1 = 2;
int numero2 = 4;

Instrucción "if" en Java

La instrucción "if" en Java es una estructura fundamental que permite tomar decisiones en función de una condición. Puedes realizar una acción específica solo cuando la condición es verdadera (true), y si la condición es falsa (false), puedes especificar una acción alternativa utilizando la estructura "if...else".

Aquí tienes un ejemplo en seudocódigo que verifica la calificación de un estudiante y muestra "Aprobado" si la calificación es mayor o igual a 60, o "Reprobado" si es menor: 


Si la calificación del estudiante es mayor o igual a 60
  Imprimir “Aprobado”
De lo contrario
  Imprimir “Reprobado”

En Java, esto se puede implementar de la siguiente manera: 


int calificacion = 65; // Debes definir previamente esta variable

if (calificacion >= 60) {
    // Acción a realizar si la calificación es mayor o igual a 60
    System.out.println("Aprobado");
} else {
    // Acción a realizar si la calificación es menor a 60
    System.out.println("Reprobado");
}

La estructura "if" en Java te permite controlar el flujo de tu programa en función de condiciones específicas. Puedes personalizar las acciones a realizar en cada rama del "if...else" para adaptarlas a tus necesidades.

Instrucción "while" en Java

En Java, la instrucción "while" se utiliza para repetir un bloque de código mientras una condición especificada sea verdadera. El bloque de código, que puede consistir en una o varias instrucciones, forma el cuerpo de esta estructura. En algún momento, la condición se vuelve falsa, por ejemplo, cuando todos los artículos de la lista de compras han sido adquiridos y eliminados. En ese punto, la repetición se detiene y se ejecuta la primera instrucción que sigue después de la instrucción "while".

A modo de ejemplo en Java, consideremos un fragmento de código diseñado para encontrar la primera potencia de 3 que sea mayor que 100. Supongamos que tenemos una variable llamada "producto" de tipo int inicializada en 3. La siguiente instrucción "while" realiza la tarea: 


int producto = 3;
while (producto <= 100) {
    producto = 3 * producto;
}

Cuando esta instrucción "while" comienza a ejecutarse, el valor de "producto" es 3. En cada iteración, la instrucción "while" multiplica "producto" por 3, lo que resulta en valores sucesivos de 9, 27, 81 y 243. Cuando "producto" finalmente alcanza el valor de 243, la condición "producto <= 100" se vuelve falsa, y la repetición se detiene. El valor final de "producto" es 243.

El método System.out.println muestra (o imprime) una línea de texto. Es decir, que imprima la cadena de caracteres contenida entre los caracteres de comillas dobles (sin incluirlas). 


public void mostrarMensaje()

Comentario


"Comentario" en Java

Un comentario que comienza con // se llama comentario de fin de línea (o de una sola línea), ya que termina al final de la línea en la que aparece. Un comentario que se especifica con // puede empezar también en medio de una línea, y continuar solamente hasta el final de esa línea. 


// Esto es un comentario

Los comentarios tradicionales (también conocidos como comentarios de múltiples líneas), como el que se muestra a continuación se distribuyen en varias líneas. Este tipo de comentario comienza con el delimitador /* y termina con */. 


/* Éste es un comentario
Tradicional. Puede
dividirse en muchas líneas */

"LLamar Procedimiento" en Java

Esta llamada hace que el método realice su tarea. Cada una de estas llamadas al método proporcionaban argumentos que especificaban los datos a mostrar. 


 public static void main( String args[] )
 {

    // llama al método mostrarMensaje
    mostrarMensaje();
 } // fin de main

El ejemplo anterior indica que el método mostrarMensaje tiene una lista de parámetros vacía; es decir, mostrarMensaje no requiere información adicional para realizar su tarea. Por esta razón, la llamada al método especifica un conjunto vacío de paréntesis después del nombre del método, para indicar que no se van a pasar argumentos al método mostrarMensaje. Cuando el método mostrarMensaje completa su tarea, el método main continúa su ejecución
